# Introduction

This is a **comprehensive documentation** site built with Next.js and MDX.

## What is MDX?

MDX allows you to write JSX in your Markdown files. This means you can:

- Import and use React components
- Write interactive documentation
- Create custom elements with full styling control

## Features

Here are some key features of this documentation:

1. **Fast Performance** - Built with Next.js 15
2. **Dark Mode** - Automatic theme switching
3. **Responsive Design** - Works on all devices
4. **Search** - Quick navigation (coming soon)
