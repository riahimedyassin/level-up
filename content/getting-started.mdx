# Getting Started
<span id="introduction"></span>

Welcome to the getting started guide! This page will help you understand the basics.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18+ installed
- npm or yarn package manager
- Basic knowledge of React and Next.js

## Installation
<span id="getting-started" > </span>

To install the project, follow these steps:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

## Configuration

You can configure the project by editing the following files:

1. **`next.config.ts`** - Next.js configuration
2. **`tailwind.config.js`** - Tailwind CSS configuration
3. **`.env.local`** - Environment variables

> **Note:** Make sure to restart the dev server after making configuration changes.
